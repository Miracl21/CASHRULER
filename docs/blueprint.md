# **App Name**: CASHRULER Mobile

## Core Features:

- Quick Expense Entry: Allow the user to quickly add an expense with title, category, amount, date, and optional note.
- Intelligent Suggestions: Implement an autosuggest system for expense titles and categories based on previous inputs. Use a LLM tool to extract the likely category from a partially input title.
- Expense List: Display a list of recent expenses, filterable by period and category.
- Savings Projects: Enable the user to create savings goals, set target amounts and deadlines, and track progress visually.
- Spending Limits & Alerts: Allow users to define daily spending limits per category and receive alerts when approaching or exceeding limits.
- Dashboard: A dashboard presenting a personalized welcome message, quick actions, daily saving challenges, and a financial overview.
- Spending Analysis: Provide visual charts (histograms) showing expense distribution by category and trends over time.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to represent money and growth in a sophisticated manner.
- Background color: Light, desaturated green (#E8F5E9) for a calm and clean backdrop.
- Accent color: Yellow-Green (#8BC34A), to provide subtle contrast with the primary color and reinforce a sense of growth.
- Clean, modern sans-serif font for readability and a professional feel.
- Use minimalist and expressive icons for expense categories and income types.
- Maintain a clean and intuitive layout with clear hierarchy and consistent spacing.
- Incorporate subtle transitions and animations to enhance user experience, such as smooth loading indicators.
